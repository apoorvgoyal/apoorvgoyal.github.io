This project aims to develop a framework for a comprehensive sustainability consulting tool that can be used by architects, designers and planners during the process of campus or neighbourhood level planning for guidance about the best practices and optimized built configuration. The tool is a plug-in for grasshopper, and using few inputs in form of rhino geometries and curves, generate optimized boundaries for the designer to start the design on. The boundary generation will be based on analysis in reference to multiple parameters such as solar orientation, wind, energy, daylight, radiation (PV) walkability etc.. The tool essentially restraints itself from a generic form finding and generates a boundary condition which informs the designer about the extents, orientation, distances etc. of the built.
